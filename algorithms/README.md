# Algorithm in C++

## 目标(goal):
1. 经典算的实现
2. 一个头文件包含一个或者一类算法，并且测试demo
3. 可以直接导入使用，便于修改源文件

## 说明(Instruction)
1. 一个投头文件.h包含一个或者一类算法或者结构放在include目录下
2. 算法的测试demo放在sample中，文件名为test_*.cpp，对应相应的头文件
3. 使用时导入对应的.h文件，如果不清楚函数的借口，建议看一下源文件
